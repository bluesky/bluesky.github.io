bluesky_queueserver_api.BItem.item_type¶

property BItem.item_type¶

The property for read-write access to the item type. Item type is a mandatory item parameter represented as a string from the list retured by BPlan.recognized_item_types.

Raises:
ValueError

Raised if the new value for item type is not in the list of recognized item types.
