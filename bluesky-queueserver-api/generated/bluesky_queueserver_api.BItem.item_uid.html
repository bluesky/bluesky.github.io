bluesky_queueserver_api.BItem.item_uid

property BItem.item_uid

The property for read-write access to the item uid. This is an optional parameter, which is typically not set by the user. In most cases the server will overwrite the UID set by the user. None is returned if UID is not set.

Raises:
TypeError

Raised if the new value is not a string.
