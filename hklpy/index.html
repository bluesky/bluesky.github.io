hklpy#

Controls for using diffractometers within the Bluesky Framework.

Based on the hkl library (documentation, source). (Caution: The GitHub source repository may not be synchronized with the latest version from repo.or.cz.)

Integrates with ophyd pseudopositioners for use with bluesky plans.
